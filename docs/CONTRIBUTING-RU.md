# Участие в разработке

## Версионирование

Пакет следует [Semantic Versioning](https://semver.org/lang/ru/).

### PATCH `1.0.X`

Обратно совместимые исправления — без изменений публичного API:

- Исправление поведения без смены сигнатур
- Расширение диапазона версий зависимости (например, разрешить новый minor-релиз)

### MINOR `1.X.0`

Новая функциональность, обратная совместимость сохранена:

- Новые artisan-команды, middleware, опции конфига (с дефолтным значением)
- Добавление поддержки новой версии Laravel или PHP

### MAJOR `X.0.0`

Ломающие изменения (breaking changes):

- Любое изменение JSON-контракта ответа (ключи, структура)
- Переименование или удаление ключей конфига
- Переименование или удаление artisan-команд или публичных классов
- Прекращение поддержки версии Laravel или PHP

### Механика тегов

- Вся разработка ведётся в ветке `main`
- Тег ставится на `main` после финального коммита:
  ```bash
  git tag v1.0.0
  git push origin v1.0.0
  ```
- Перед тегом обновить `CHANGELOG.md`
